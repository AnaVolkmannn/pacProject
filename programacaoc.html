Linguagem de programação em C

Nessa página você terá acesso a todo o conteúdo referente a linguagem de programação em C, para a apronfundar e alavancar seus estudos.

Introdução a linguagem C

A linguagem de programação é como falar com um computador, mas não se trata de uma conversa cotidiana como fazemos entre pessoas. É mais sobre fornecer instruções detalhadas sobre o que queremos que o computador execute. Com a linguagem C, podemos fazer o computador realizar diversas tarefasm como realizar cálculos, exibir mensagens na tela e até mesmo desenvolver jogos. É uma linguagem que nos permite criar muitas coisas no mundo da computação. A linguagem C é uma linguagem de uso geral, o que significa que é adequada para quase qualquer tipo de projeto. Além disso, ela é altamente portátil e oferece uma execução extremamente rápida.

Como usar

O primeiro passo é colocar a biblioteca “stdio” que nos da acesso a entrada e saída de dados, e fornece também o “printf()”. Logo após, vamos utilizar a biblioeca “stdlib”, que acessa a função de gerenciar como a memória é reservada e liberada durante a execução de um programa.


Após as bibliotecas, vem a função “int main(void)” , que é a entrada de dados da linguagem C. O “int” representa que ela sempre irá retornar a um número inteiro. Toda a função tem um corpo, e ele deve ser envolvido com chaves ({), e o que vai dentro do corpo é tudo que o precisa para realizar todas as operações.
