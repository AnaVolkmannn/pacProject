Laços de Repetição

  1. Escreva um programa em C que imprima os números de 1 a 10 usando um laço ‘for’. (Fácil)

  2. Escreva um programa em C que peça ao usuário para digitar um número inteiro e depois imprima os números de 1 até o número digitado usando um laço ‘while’ (Fácil)

  3. Escreva um programa em C que leia 5 números inteiros do usuário e imprima a soma desses números usando um laço ‘for’ (Médio)

  4. Escreva um programa em C que peça ao usuário para digitar um número inteiro e depois imprima todos os números pares de 1 até o número digitado usando um laço do ‘while’ .(Médio)

  5. Escreva um programa em C que leia um número inteiro n e imprima os primeiros n números da sequência de Fibonacci usando um laço ‘for’ (Difícil)

  6. Escreva um programa em C que leia um número inteiro e verifique se ele é um número primo usando um laço ‘for’. O programa deve imprimir uma mensagem indicando se o número é primo ou não. (Difícil)
