Aula 3:

Condicionais e Laços de Repetição

Nessa aula você pode ter acesso á várias explicações que irão te auxiliar a aprender mais sobre estruturas condicionais e laços de repetição em linguagem C.

Como funcionam as estruturas condicionais?

Na programação em C, as estruturas condicionais são usadas para tomar decisões com base em condições específicas. A linguagem C oferece várias formas de expressar condições, sendo as mais comuns: if, else if, else e switch

Como entender laços de repetição

Os laços de repetição são usados em programação para executar um bloco de código várias vezes. Em C, os laços de repetição mais comuns são for, while e do-while.

Ambas se complementam?

Estruturas condicionais e os laços de repetição trabalham juntos para automatizar e personalizar o processo de um programa, economizando tempo e esforço enquanto atendem às necessidades.

INTRODUÇÃO A ESTRUTURAS CONDICIONAIS

Uma estrutura condicional é uma construção de programação que permite que o código tome decisões e execute diferentes blocos de instruções com base em determinadas condições. As estruturas condicionais verificam uma ou mais expressões lógicas (que podem ser verdadeiras ou falsas) e, dependendo do resultado, direcionam o fluxo de execução do programa para diferentes partes do código.

Estruturas Condicionais em Linguagem C

Estrutura If

A estrutura if é usada para executar um bloco de código se uma condição especificada for verdadeira.

Exemplo em código:

#include <stdio.h>
int main() {
int x = 10;
if (x > 0) {
printf("x é positivo.\n");
}
return 0;
}

Estrutura if-else

A estrutura if-else permite executar um bloco de código se a condição for verdadeira e outro bloco se a condição for falsa.

Exemplo em código:

#include <stdio.h<
int main() {
int x = -10;
if (x > 0) {
printf("x é positivo.\n");
} else {
printf("x não é positivo.\n");
} return 0;
}

Estrutura do switch

A estrutura switch é usada quando você precisa selecionar uma entre muitas alternativas com base no valor de uma variável.

Exemplo em código:

#include <stdio.h<
int main() {
int dia = 3;
switch (dia) {
case 1: printf("Segunda-feira\n"); break;
case 2: printf("Terça-feira\n"); break;
case 3: printf("Quarta-feira\n"); break;
case 4: printf("Quinta-feira\n"); break;
case 5: printf("Sexta-feira\n"); break;
case 6: printf("Sábado\n"); break;
case 7: printf("Domingo\n"); break;
default: printf("Dia inválido\n");
} return 0;
}
